# Future Features Roadmap

## Phase 2: Enhanced Local Experience
- **Problem Deduplication**: Use Problem IDs instead of URLs to ensure accuracy even if a user visits a problem via different links.
- **Streak Tracking**: Visual "flame" icons in the popup to show consecutive days of coding.
- **Charts & Graphs**: Integrated Bar/Pie charts showing platform distribution (e.g., 60% LeetCode, 40% Codeforces).

## Phase 3: Cloud & Sync
- **Supabase Integration**: Move from local storage to a Supabase database for cross-browser syncing.
- **User Authentication**: Allow users to log in to save their history forever.

## Phase 4: Social & Notifications
- **Discord Bot Integration**: Automatically post a "Solved!" message to a specific Discord channel via Webhooks.
- **Native Notifications**: Browser-level "Toast" notifications when a problem is successfully logged.
- **Custom Tags**: Allow users to manually add tags (e.g., #DP, #Graph) to solved problems in the popup.
